Project Overview

This project is a Java-based quantum computing simulator that demonstrates quantum computing fundamentals through implementations of the Deutsch-Jozsa and Bernstein-Vazirani algorithms.

Features

  • Complex Number Arithmetic
  • Quantum State Simulation
  • Hadamard Gate Implementation
  • Oracle Functions
  • Quantum Measurement
  • Deutsch-Jozsa Algorithm
  • Bernstein-Vazirani Algorithm
  • Statistical Testing

Quantum Concepts

Qubits

Classical computers use bits that are either 0 or 1. Quantum computers use qubits. 

α|0⟩ + β|1⟩

Superposition

A qubit can represent multiple possibilities simultaneously.

Interference

Quantum amplitudes can reinforce or cancel one another. This is what gives many quantum algorithms their power.

Measurement

Measurement collapses a quantum state into one definite outcome.
